# Proto 301 Content Command Center

A Vite + React dashboard for tracking content metrics, built with TypeScript, Material UI, and Recharts.

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:5173` after the dev server starts.

## Project structure

- `src/`
  - `App.tsx` — main dashboard UI and chart layout
  - `main.tsx` — app entry point
  - `styles.css` — global page styling
  - `types.ts` — shared TypeScript types
  - `data/content_metrics.json` — sample metric data
- `scripts/` — helper scripts for generating sample data
- `package.json` — dependencies and npm scripts
- `tsconfig.json`, `tsconfig.node.json` — TypeScript configuration
- `vite.config.ts` — Vite build configuration

## Notes

- `dist/` is generated by the build process and is excluded from version control.
- `Brief.MD` contains the project brief and notes.
